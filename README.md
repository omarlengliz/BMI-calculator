#  BMI Calculator

A simple   BMI calculator app using Flutter
# How It Works

 1. Run cd   BMI-calculator  command on your terminal 
 2. Run  dart pub get command on your terminal  to get all dependencies  used  
 3. Run flutter run on your terminal
